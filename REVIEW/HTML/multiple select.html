1. ctrl + d : 현재 커서가 위치한 단어를 선택하고, 일치하는 다음 단어도 하나씩 선택
2. ctrl + shift + L : 커서가 위치한 단어를 한번에 모두 선택
Shift + Tab = 선택 부분 좌측으로 한 칸 들여쓰기
CTRL + SHIFT + L : CTRL + F로 단어 찾은 뒤에, 본문에서 단어를 가리키고 이 단축키를 누르면 찾은 단어들 모두 선택해줍니다. 동시 편집, 복사 가능합니다.
CTRL + ALT + UP / DOWN: 여러 줄 동시 편집 할 수 있도록 커서를 더해줍니다. (No.1 과 약간 비슷합니다.)
